# **CodeAlpha_HangmanGame**

## **Project Description**
This is a **text-based Hangman game** developed as part of the **CodeAlpha Python Programming Internship**. The objective of the game is for a player to guess a hidden word one letter at a time.

## **Task Requirements (Task 1)**
As per the internship instructions, this project adheres to the following simplified scope:
*   **Predefined Word List:** A list of 5 predefined words (Bamboozle, Shenanigans, Skedaddle, Kerfuffle, and Hullabaloo).
*   **Incorrect Guess Limit:** Players are allowed a maximum of **6 incorrect guesses** before the game ends.
*   **User Interface:** Utilises basic console input and output without the need for graphics or audio.

## **Key Concepts Used**
The implementation demonstrates proficiency in the following Python fundamentals required for the task:
*   **`random` module**: Used to select a word randomly from the predefined list.
*   **`while` loop**: To manage the game loop and continuous user interaction.
*   **`if-else` logic**: To handle correct and incorrect guesses, as well as game-over conditions.
*   **Strings and Lists**: For managing word display, underscores, and the history of guessed letters.

## **Features**
*   **Input Validation**: The script ensures only single letters are accepted and warns the user if they repeat a guess.
*   **Real-time Progress**: Displays the current status of the word using underscores and reveals correctly guessed letters.
*   **Game Status**: Shows the remaining number of attempts and a list of letters already guessed.

## **How to Run**
1. Ensure Python 3.x is installed on your system.
2. Clone this repository to your local machine.
3. Run the script using: `python hangman.py`
